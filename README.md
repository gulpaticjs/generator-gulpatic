# ![Gulpatic](https://raw.githubusercontent.com/boriskaiser/gulpatic/demo/resources/logo.png)
> Yeoman generator for Gulpatic


## Getting started
```bash
  npm install -g yo
  npm install -g generator-gulpatic
  mkdir MyAppName
  cd MyAppName
  yo gulpatic
  npm start
```
