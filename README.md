## Usage
In case you don't have `yo`:
```bash
npm install -g yo
```

Get `generator`:
```bash
npm install -g generator-gulpatic
```

Scaffold your app:
```bash
mkdir myAppName
cd myAppName
yo gulpatic
npm start
```
